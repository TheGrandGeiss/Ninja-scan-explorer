# Ninja Scan Explorer

A fast blockchain explorer focused on Solana token and wallet intelligence. Ninja Scan Explorer helps users inspect wallets, analyze tokens, view recent transactions, and understand on-chain activity through a clean frontend built with Next.js.

## Live Demo

**[https://ninja-scan-explorer.vercel.app](https://ninja-scan-explorer.vercel.app)**

---

## Features

* 🔎 Search Solana wallet addresses
* 🪙 Search token mint addresses
* 📈 Token analytics dashboard
* 📊 Price chart visualization
* 💸 Recent transaction tracking
* 🤖 AI-generated token risk / summary insights
* ⚡ Fast UI with modern Next.js App Router
* 📱 Responsive design

## Getting Started

### 1. Clone the project

```bash
git clone https://github.com/your-username/ninja-scan-explorer.git
cd ninja-scan-explorer
```

### 2. Install dependencies

```bash
npm install
```

### 3. Add environment variables

Create a `.env.local` file:

```env
BIRDEYE_API_KEY=your_key_here
GROQ_API_KEY=your_key_here
```

### 4. Run development server

```bash
npm run dev
```

Open `http://localhost:3000`
